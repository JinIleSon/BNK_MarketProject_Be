로그인 도움말

아이디·비밀번호 찾기, 자동 로그인, 소셜 로그인, 보안 안내까지 한 곳에서 확인하세요.

빠른 해결

아이디 찾기 가입 시 입력한 정보(이름·이메일/휴대폰)로 아이디를 확인합니다. 비밀번호 재설정 본인 확인 후 임시 비밀번호 발급 또는 새 비밀번호를 설정합니다. 고객센터 문의 1:1 문의 또는 전화 상담(대표번호 080-123-4567)으로 도움을 받아보세요.

자주 묻는 질문(FAQ)

Q1아이디/비밀번호가 기억나지 않아요.
• 아이디 찾기에서 본인정보로 아이디를 확인하세요.
• 비밀번호 재설정에서 본인 확인 후 임시 비밀번호를 발급받거나 새 비밀번호를 설정할 수 있어요.
임시 비밀번호 로그인 후 비밀번호 변경을 꼭 진행하세요.
Q2자동 로그인이 안 되거나 자꾸 해제돼요.
자동 로그인은 브라우저 쿠키에 저장됩니다.
1) 쿠키/사이트 데이터 차단 해제 여부
2) 시크릿(Incognito) 창 사용 여부
3) 보안 프로그램/확장 기능이 쿠키를 삭제하지 않는지
4) 여러 기기 동시 로그인 시 보안상 해제될 수 있어요.
Q3카카오/네이버/구글 소셜 로그인이 안 돼요.
1) 각 서비스 정상 로그인 여부 확인
2) 최초 1회 NICHIYA 계정 연동 동의 완료 필요
3) 팝업 차단 해제(동의 창이 차단될 수 있음)
4) 계속 실패 시 고객센터로 오류 시각/메시지를 알려주세요.
Q4계정이 잠겼다는 메시지가 나와요.
실패 횟수 초과 시 일시 잠금됩니다. ① 약 10분 후 자동 해제 또는 ② 비밀번호 재설정으로 즉시 해제 가능.
Q5회원가입 인증 이메일/문자가 오지 않아요.
스팸함 확인 → 인증코드 유효시간(10분) 경과 여부 → 수신 차단 해제 후 재요청. 지속 실패 시 대표번호(080-123-4567)로 문의해주세요.
Q6기업/공용 PC에서 안전하게 로그인하려면?
• 사용 후 반드시 로그아웃
• 다른 사이트와 다른 비밀번호 사용, 주기적 변경
• 주소창이 nichiya.co.kr인지 확인(피싱 주의)
• 2단계 인증 사용 시 개인 기기에만 설치
Q7오류 화면(500/404 등)이 나와요.
새로고침/캐시삭제 후 재시도 → 다른 브라우저/네트워크 시도 → 지속 시 고객센터로 캡처와 발생 시각 전달.

추가 안내
