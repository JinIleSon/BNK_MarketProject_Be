# NICHIYA (日夜)
### High-End Select Shop — Where Night Meets Day

**NICHIYA**는 ‘밤낮(日夜)’이라는 이름처럼,  
시간의 경계를 넘어 언제 어디서나 즐길 수 있는 하이엔드 온라인 편집숍입니다.  
세련된 감각과 품격 있는 브랜드를 엄선하여,  
하루의 모든 순간에 어울리는 라이프스타일을 제안합니다.

> **밤에도, 낮에도 — 당신의 취향은 멈추지 않습니다.**

### 🛍 프로젝트 개요
**BNK_MarketProject_Be**  
NICHIYA는 Spring Boot 기반의 전자상거래 플랫폼으로,  
사용자가 밤낮없이 고품질 제품을 탐색하고 구매할 수 있도록 설계되었습니다.  
관리자(Admin)와 사용자(User) 페이지를 분리하여  
효율적인 상품 관리와 유연한 UI/UX 경험을 제공합니다.

---

### ⚙️ 주요 기술 스택
- **Backend:** Spring Boot 3.x, Java 17, MyBatis, QueryDSL  
- **Database:** Oracle XE (XEPDB1)  
- **Frontend:** Thymeleaf, JavaScript, CSS  
- **CI/CD:** GitHub Actions, AWS EC2
